# DRC App (React + Vite)

This project has been migrated from Create React App to [Vite](https://vitejs.dev/) for faster compilation and development. 

The application is deployed to **Google App Engine Standard** (`nodejs22`).

## Development Scripts

To avoid accidentally connecting to the production Firebase database while testing locally, always use the automated PowerShell scripts provided in the root directory.

### Local Development
To start the local development server (with auto-reloading):
```powershell
.\start-local.ps1
```
*(This script will automatically copy `src/cred/firebase-test.js` to `src/cred/firebase.js` before starting Vite on `localhost:3000`)*

### Production Deployment
To build and deploy the application to Google Cloud App Engine:
```powershell
.\deploy-prod.ps1
```
*(This script will copy `src/cred/firebase-prod.js` to `src/cred/firebase.js`, build the optimized production assets using Vite, and run `gcloud app deploy` automatically)*

---

## Technical Details
- **Routing**: `app.yaml` is configured to natively serve the `build/` directory generated by Vite.
- **Dependencies**: React, React Router, React Bootstrap, Firebase.
- **Node Version**: 22+
